Ayuda

Esta es la sección de Ayuda, las demás secciones de la aplicación son:

  • Ver Estado: Esta sección del menu muestra el estado de conexión con el dispositivo HeartSense.
  • Iniciar Holter: Esta sección del menu permite inciar el estudio.
  • Analizar Datos: Esta sección del menu, te permite ver el gráfico de los resultados del estudio
  • Borrar Datos: Esta sección del menu, te permite borrar el cotenido en la memoria del dispositivo
Si usted necesita asistencia, por favor contacte nuestro equipo de soporte. https://github.com/HeartSense-HolterMonitor-ACySE-UNRN

Algunos Error comunes:

  • Error: HolterNotFound -> Significa que el dipositivo no se encuentra conectado correctamente. Por favor revise las conexiones y vuelva a intentar Si el error persiste, pruebe reiniciar la app, si aún así no logró que funcione contactese con nuestro equipo.
