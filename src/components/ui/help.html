HEARTSENSE Monitor Holter Help

Welcome to the help section of HEARTSENSE Monitor Holter. This application allows you to monitor and analyze heart rate data. Here are the main features of the application:

  • Estado del Holter: This menu item displays the current status of the holter device.
  • Borrar SD: Use this menu item to delete data from the SD card.
  • Iniciar Holter: Click on this menu item to start the holter monitoring process.
  • Analizar Datos: This menu item provides data analysis and visualization of the holter study using Plotly.
If you need further assistance, please contact our support team.
